**📊 Sales Analysis Dashboard (Excel Practice)**

📌 Overview

This Excel practice exercise focuses on performing structured sales data analysis using Microsoft Excel.
The objective was to work with a raw dataset, clean and transform the data, apply analytical functions, and finally build an interactive dashboard to visualize key business metrics.

This practice helped in strengthening real-world Excel skills used in data analysis roles.
